# PhotoApp
Modern Android Photo Editing Application built with Java, MVVM Architecture, Room Database, ML Kit Face Detection, and Material Design. 

## Features

* Gallery image selection
* Camera image capture
* Image cropping
* Face detection using ML Kit
* Photo editing workflow
* Local data storage using Room Database
* Material Design UI
* ViewBinding support
* Clean MVVM Architecture

## Tech Stack

* Java
* Android SDK
* MVVM Architecture
* Room Database
* LiveData
* ViewModel
* ML Kit Face Detection
* Glide
* Navigation Component
* Material Design Components

## Architecture

The application follows a clean MVVM architecture with proper separation of concerns:

* UI Layer
* ViewModel Layer
* Repository Layer
* Local Database Layer

## Requirements

* Android 7.0 (API 24) and above
